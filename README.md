# Tic Tac Toe Game (GUI)

A classic, interactive, and user-friendly **Tic Tac Toe** (X-O) game built using Python and the **Tkinter** library for the Graphical User Interface (GUI).

---

## 🚀 Features
* **Interactive GUI:** Clean and simple graphical interface built with Tkinter.
* **Turn Indicator:** Clearly shows whose turn it is (Player X or Player O).
* **Auto-Detection:** Automatically detects wins (horizontal, vertical, diagonal) and draws.
* **Reset Button:** Easily restart the game at any time to play again.
* **Responsive Layout:** Centered buttons that pop up notifications upon winning.

---

## 🛠️ Technologies Used
* **Python 3**
* **Tkinter** (Python's standard GUI library)

---
